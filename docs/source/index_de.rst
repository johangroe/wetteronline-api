================
WetterOnline API
================

**wetteronline** ist ein Python Modul, welches inoffiziellen Zugriff auf die Wetterdaten von `wetteronline.de <https://www.wetteronline.de>`__.
Es benutzt webscraping und bietet (hoffentlich) eine *simple* und *intuitive* API.

.. note::

   Dieses Projekt ist unter aktiver Entwicklung. (Die Dokumentation ebenfalls!)
   Wäge gut ab, ob du bereits Code welcher auf dieses Modul baut schreibst, denn Funktionen und Attribute könnten sich jederzeit ändern.
